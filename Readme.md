# To-Do List

MERN Stack

## Prerequisites
 - Node.js
 - MongoDB

## Installation

Create mongoDB
```
cd /api/dist/migrations
node createDb.js
```

Create modules migrations
```
cd /api/dist/modules/{module name}/migrations
node {migration_file_name}.js
```
## API
Install packages
 ```sh
cd /api/
npm install
```

Gulp watch
 ```sh
cd /api/
gulp
```

Start a development server
 ```sh
cd /api/
npm run dev
```
API is running at http://localhost:3000/

## Web
Install packages
 ```sh
cd /web/
npm install
```

Gulp watch
 ```sh
cd /web/
gulp
```

Start a development server
 ```sh
cd /web/
npm run dev
```
Front-end is running at http://localhost:8001/
